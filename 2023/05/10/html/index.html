HTML:结构层，从语义的角度去搭建网页结构

1.HTML简介

  1. 什么是网页？

    网页由三部分组成：内容、表现、行为

    • 内容：是页面数据。例如文本、数据库向前端展示的数据等都称为内容–而内容是通过HTML技术来展示

    • 表现：是这些内容在页面上的展示形式，例如：布局、颜色、大小等–一般使用CSS技术实现

    • 行为：是指页面中元素与输入设备交互的响应，例如鼠标点击按钮的响应行为等–一般使用JavaScript来实现

  2. 什么是HTML？

    HTML：超文本标记语言(Hyper Text Markup Language)

    通过标签来标记要显示的网页中的各个部分，网页本身也是一种文本文件 通过标记符，告诉浏览器如何显示其中的内容，例如文字如何处理，画面如何排版等

3.HTML元素

HTML文档是由HTML元素构成的

什么是HTML元素？

由一个起始标签开始到该终止标签结尾这个整体称为一个HTML元素

4.HTML标签

HTML元素是由HTML标签定义的

什么是标签？

  • HTML标签是用来标记HTML元素的

  • HTML标签被<>符号包裹 HTML的标签是成对出现的，前面的称为起始标签 ，后面的称为终止标签

  • HTML标签对大小写不敏感，但是建议使用小写，尽量不要使用大写–不符合HTML设计规范

常见基本标签

标签 描述
用于告知浏览器使用哪种HTML规范(定义文档类型)
< html> 定义HTML文档
< head> 定义HTML的头信息
< body> 定义文档的主体
< h1> to < h6> 定义标题1->标题6
< p> 定义段落
< br> 插入换行符
< hr> 定义水平线
< !—-> 定义注释
< div> 定义文档中的块
< title> 定义文档的标题
< font> (不赞成使用了–建议使用CSS)定义文字的字体、尺寸和颜色
< ul> 定义无序列表。
< select> 定义选择列表
< ol> 定义有序列表。

4.1HTML属性

属性是用于为HTML元素提供附加信息，属性的格式为：name=“value”

属性是依赖于标签存在的

contenteditable设置是否允许用户编辑元素，true表示允许，由该标签包裹的内容都允许被编辑

9.文本域

标签是textarea，是一个双标签，文本级标签 属性值(rows和cols)

rows：定义文本域的可视区域有几行，单位是数字

cols：当前行显示的字节数量(以英文为主)，单位数字

placeholder：占位符

textarea默认是可以通过拖动右下角实现放大或者缩小文本域的，我们可以通过设置style的resize属性，值为none，去掉可拖拽区域

11.注释

在代码中有一些特殊结构，特点是在编辑器能看到，在浏览器中看不到，这种结构为注释，特点是浏览器不加载。

作用：用来注释代码，添加合理的解释从而实现对代码的可读性；先注释掉一部分不使用的代码，便于后期恢复用于代码调试
